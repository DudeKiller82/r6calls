R6Calls.com About

Purpose

R6Calls.com is designed to help players to learn Rainbow Six Siege callouts on maps. The official callouts and the players callouts. It only refer to English language as this is the most spoken language in the community. For a better understanding it shoud be the only language in game.

It can be used as a reference tool on a second device while playing, or even in the UPlay or Steam overlay browsers on PCs. It was originaly based on the architecture of R6Maps.com

Some images are taken from Ubisoft's Site, and then further edited.

Site Help

Keyboard Shortcuts

You can switch between floors using keyboard shortcuts.  For example:

  • 0 or tilde Basement
  • 1 1st Floor
  • 2 2nd Floor
  • 3 Roof

You can toggle the following:

  • L Learning Mode
